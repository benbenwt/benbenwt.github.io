### debug

##### 捕获特定异常，自动设置断点

```
点击debug，点击左侧的两个圆形(view breakpoints)，添加想捕获的异常，然后点击done，再debug。我们就会自动到达异常触发点，再观察此时的变量值即可。
在左侧栈frame选择触发的栈帧级别，右侧观察栈中变量。
```
